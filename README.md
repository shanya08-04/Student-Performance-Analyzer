Student Performance Analyzer

##Overview
The Student Performance Analyzer is a Python-based project that analyzes students' academic performance using a dataset of marks. It calculates key metrics and generates graphical representations to help understand performance trends.

This project is designed as part of the Bring Your Own Project (BYOP) and focuses on solving a real-world academic analysis problem without using machine learning or any UI.

##Features
+Calculates total and average marks of each student
+Identifies the top performer
+Generates graphical visualizations:
 +Average marks of all students (Bar Graph)
 +Subject-wise average marks (Bar Graph)
 +Individual student performance (Line Graph)
+ Allows user to view detailed performance of a specific student

# Technologies Used
+Python
+pandas
+matplotlib
